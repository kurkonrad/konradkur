```shell
$ npm install

```
# start project
```shell
$ gulp dev
```
# build cv
```shell
$ sudo apt install wkhtmltopdf
$ cd summary
$ wkhtmltopdf templates/konrad_kur_cv.html dist/konrad_kur_cv.pdf
```
